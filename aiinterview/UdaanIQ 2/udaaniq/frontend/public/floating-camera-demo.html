Floating Camera Component Demo

How to Test the Floating Camera

This demo shows the features of the updated FloatingCamera component that fixes the findDOMNode error:

Key Features Implemented:

  • ✅ Fixed findDOMNode error by using nodeRef API in react-draggable
  • ✅ Draggable camera preview that can be moved around the screen
  • ✅ Double-click to toggle between small (160x120) and large (320x240) sizes
  • ✅ Mute/Unmute button for audio control
  • ✅ Close button to hide the camera preview
  • ✅ Position and size persist in localStorage
  • ✅ Material 3 styling with rounded corners and shadow
  • ✅ Accessibility features (aria-label, proper contrast)
  • ✅ Compatible with React 18 and Next.js

To Test in the Actual Application:

  1. Start the backend server: cd backend && npm run dev
  2. Start the frontend server: cd frontend && npm run dev
  3. Navigate to Mock Interview or Proctored Interview
  4. Click "Start" to begin the interview
  5. Grant camera permissions when prompted
  6. Observe the floating camera preview that appears
  7. Try dragging it around the screen
  8. Double-click to change its size
  9. Use the mute and close buttons
  10. Refresh the page to see that position/size are preserved
